[B]Version 2.0.66 - 04 Jul 2026[/B]
- Base Code Changes
    - Reverted reuselanguageinvoker to false. With it enabled, addon
      code updates did not take effect until Kodi was fully restarted
      (the interpreter stayed resident with old code). For a scraper
      addon that hits the network on every click, the small navigation
      speedup was not worth updates silently not applying.

[B]Version 2.0.65 - 04 Jul 2026[/B]
- Base Code Changes
    - Removed the ResolveURL button from addon Settings: Kodi cannot
      reliably open one addon's settings from inside another's settings
      dialog (it rendered ResolveURL's login fields into the wrong page).
      Use the "ResolveURL / Debrid Settings" item in the main menu instead,
      which opens the resolver config cleanly.

[B]Version 2.0.64 - 04 Jul 2026[/B]
- Base Code Changes
    - Added direct "ResolveURL / Debrid Settings" shortcut - opens the
      ResolveURL config (TorBox, RealDebrid, etc) from both the main
